# File Tree Component Example

The FileTree component allows you to display interactive directory structures in your documentation, similar to what you see in the [Terragrunt documentation](https://terragrunt.gruntwork.io/docs/guides/terralith-to-terragrunt/step-5-adding-terragrunt/).

## Basic Usage

Here's a simple project structure:

<FileTree data={[
  {
    name: 'my-project',
    type: 'directory',
    children: [
      { name: 'src', type: 'directory', children: [
        { name: 'index.js', type: 'file' },
        { name: 'utils.js', type: 'file' }
      ]},
      { name: 'package.json', type: 'file' },
      { name: 'README.md', type: 'file' }
    ]
  }
]} />

## File Tree with Inline Content

You can also show file contents inline by setting `showContent={true}`:

<FileTree 
  data={[
    {
      name: 'terraform-module',
      type: 'directory',
      children: [
        { 
          name: 'main.tf', 
          type: 'file',
          content: `resource "aws_vpc" "main" {
  cidr_block = var.cidr_block
  
  tags = {
    Name = var.name
  }
}`,
          language: 'hcl'
        },
        { 
          name: 'variables.tf', 
          type: 'file',
          content: `variable "cidr_block" {
  description = "CIDR block for VPC"
  type        = string
  default     = "10.0.0.0/16"
}

variable "name" {
  description = "Name of the VPC"
  type        = string
}`,
          language: 'hcl'
        }
      ]
    }
  ]} 
  showContent={true}
/>

## Component Features

The FileTree component includes:

- **Interactive Navigation**: Click directories to expand/collapse
- **File Type Icons**: Different icons for different file types
- **Syntax Highlighting**: Code content is syntax highlighted
- **Responsive Design**: Works on all screen sizes
- **Dark Mode Support**: Automatically adapts to theme
- **Accessibility**: Keyboard navigation and screen reader support

## Use Cases

This component is perfect for:

- **Tutorials**: Show project structure in step-by-step guides
- **Documentation**: Explain how to organize files and directories
- **Code Examples**: Display file contents with proper syntax highlighting
- **Onboarding**: Help new developers understand project layout
- **Migration Guides**: Show before/after directory structures

## Implementation

The FileTree component is built with React and TypeScript, making it easy to integrate into any Docusaurus site. It uses CSS modules for styling and supports all modern browsers.
